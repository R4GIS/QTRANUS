Results

This part of the program is used to produce maps with the results of a TRANUS model. In this manual we are using the model of Swindon as an example. This model and all related data may be downloaded from:

TRANUS site

To begin the process, a window is presented with boxes to fill:

Layers group name

Give a name to put all the layers that you will produce into a group - Obligatory field

TRANUS workspace

A TRANUS project resides in a unique folder in your computer. As soon as you specify a valid TRANUS workspace, you will notice that the Scenario Tree appears with valid information in the left panel. This is because QTRANUS looks in the folder you specified for a file called TRANUS.CTL. This is the file where the scenario tree is specified, a fundamental structure in TRANUS modeling. You may click on the little + to expand the scenario tree.

Zones shapefile

Specify the shape file where the polygons of the zones are contained. You may use the button provided to explore your pc. As soon as you put a valid shape file name, a map with the polygons should appear as a background. The shapefile must contain one polygon for each zone in the model. Make sure that this is so.

Next, Select zones shape Id with a pull-down list of all the fields in the zones shapefile that you specified. Select the one that contains the Zone Id numbers. This must be integer values exactly matching the Zone Id identifiers in the model.

If no Zone shapefile is specified, the program will not be able to produce maps based on zonal information.

The other two boxes, Network Shapefile and Centroid Shapefile correspond to parts of the program that are still under development.

At the bottom there are three buttons called ‘Zones’, ‘Network’ and ‘Matrix’. They are all inactive. As soon as a valid Zones Shapefile is specified, the Zones and the Matrix buttons become active. The Network option is not available in this version. Press the Zones button, and another window appears to generate QGIS layers with thematic zone-based results, as described below. Press the Matrix button and another window will appear to generate flow-maps from trip matrices.

Once one or more map or matrix layers have been defined, you may save the QGIS project in the usual way, doing File-Save or File-Save As and giving it a name. This means that the next time you open the QGIS project, all your maps will be there exactly as you left them. You may then add more layers to the project using the QTRANUS plugin again and again.
